# GA-Login
使用二次验证的登陆系统。

## 项目介绍

登陆系统连接MySQL数据库，数据库中保存用户名和加盐Hash后的值。
登陆后用户可以在用户设置中设置双因子认证，这里使用的是Google Authenticator
